# Setup

## Install

```sh
pip3 install -r requirements.txt
```

## env

create a .env file like so:

```
DOLPHIN_USERNAME=EPITA_GROUPE5
DOLPHIN_PASSWORD=pwd
```

# Launch

## Network demo

```sh
python3 network.py
```
